# Overview
This site is intended to onboard new users to **mess2** (inspired by the work of **[MQP RVC2401](https://localhost:3000)**), a collection of packages integrated with **ROS2** that support the **ACE Lab's CSCP Research**. Lab hardware is discussed when necessary, but the primary focus of this site is software, specifically how theory is implemented into ROS2.

## Introduction to ROS2
Robot Operating System 2 (ROS2) is a middleware consisting of open-source libraries and tools that are commonly used in robotics research and applications. Understanding how ROS2 works is essential for understanding how the mess2 software suite works. If you are not familiar with ROS2, please complete all of the tutorials in ROS2 Introduction. If you are familiar with ROS2, you may skip this section, but consider reviewing the tutorials as a refresher.
