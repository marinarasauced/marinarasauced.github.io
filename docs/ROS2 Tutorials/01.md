# Ubuntu 24.04 Installation

## Overview
To use ROS2 on your local machine, you must have access to a Linux OS. It is prefered to have a dedicated Ubuntu installation or to dual boot Ubuntu with Windows; however, it is possible to run ROS2 in a virtual machine. These instructions walk you through installing Windows Subsystem for Linux (WSL) and creating an Ubuntu 24.04 virtual machine. If you have a dedicated Linux OS, skip this section.

:::warning
The following instructions are for Windows machines only. MacOs users can follow subsequent sections after installing Ubuntu via alternative means.
:::

:::warning
These instructions do not apply to any lab hardware that runs a dedicated Linux OS.
:::
