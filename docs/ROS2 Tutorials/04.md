# ROS2 Jazzy Installation

## Overview
The following installation instructions detail how to install ROS2 Jazzy Jalisco from the Debian Packages on Ubuntu 24.04 Noble Numbat. More information about the installation can be found [here](https://docs.ros.org/en/jazzy/Installation/Ubuntu-Install-Debians.html).

## Installation

### Set Locale
Ensure that you have a locale that supports `UTF-8` by running the following command:

```zsh
locale
```

If you do not have a locale that supports `UTF-8`, update your locale.

### Add ROS2 Apt Repository
Enable the Ubuntu Universe repository:

```zsh
sudo apt install software-properties-common
sudo add-apt-repository universe
```

Add the ROS2 GPG key with apt:

```zsh
sudo apt update && sudo apt install curl -y
sudo curl -sSL https://raw.githubusercontent.com/ros/rosdistro/master/ros.key -o /usr/share/keyrings/ros-archive-keyring.gpg
```

Add the repository to your sources list:

```zsh
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/ros-archive-keyring.gpg] http://packages.ros.org/ros2/ubuntu $(. /etc/os-release && echo $UBUNTU_CODENAME) main" | sudo tee /etc/apt/sources.list.d/ros2.list > /dev/null
```

### Install Development Tools
Install the development tools so that you can build ROS2 packages:

```zsh
sudo apt update && sudo apt install ros-dev-tools
```

### Install ROS2
Update your apt repository cache and update your system:

```zsh
sudo apt update && sudo apt upgrade
```

Install the desktop version of ROS2:

```zsh
sudo apt install ros-jazzy-desktop
```

### Source Environment
Source the `setup.zsh` file in the ROS2 installation path to enable ROS2 in the current terminal.

```zsh
source /opt/ros/jazzy/setup.zsh
```

This command only sources `setup.zsh` in the current terminal. Run the following command to modify the `.zshrc` file so that every time a new terminal is opened, `setup.zsh` is automatically sourced:

```zsh
echo "
# ROS2 Installation
source /opt/ros/jazzy/setup.zsh" >> ~/.zshrc
```

### Initialize Dependencies
Initialize ROS2 dependencies:

```zsh
sudo rosdep init
rosdep update
```

### Test Installation
Open a new terminal and run the following command:

```zsh
ros2 run demo_nodes_cpp talker
```

Open another new terminal and run the following command:

```zsh
ros2 run demo_nodes_py listener
```

As per [docs.ros.org](https://docs.ros.org/en/jazzy/Installation/Ubuntu-Install-Debians.html), "you should see the `talker` saying that it's `Publishing` messages and the `listener` saying `I heard` those messages. This verifies both the C++ and Python APIs are working properly."
